# 🌐 Meus Sites Favoritos

Projeto desenvolvido em HTML com o objetivo de criar uma página web simples contendo alguns dos meus sites favoritos.

## 📚 Sobre o projeto

A página apresenta:

- Nome dos sites
- Logos/imagens de cada site
- Links para acesso
- Estrutura básica em HTML5

## 🚀 Sites utilizados

- YouTube
- Google
- Instagram
- Netflix
- Spotify

## 🛠️ Tecnologias utilizadas

- HTML5

## 📷 Preview do Projeto

A página contém:

- Título principal
- Imagens clicáveis
- Links externos
- Separação visual utilizando `<hr>`

## 📂 Estrutura do Projeto

```bash
📁 meus-sites-favoritos
 ├── index.html
 └── README.md
```
